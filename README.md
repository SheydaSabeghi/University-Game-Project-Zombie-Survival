## About the Game
Zombie Survival is a single-player university game project where the player controls a zombie character.  
The goal of the game is to eliminate enemies, overcome obstacles, collect score bonuses, and reach the final scene to win.

## Gameplay Overview
- Player character: Zombie
- Single-player shooter gameplay
- Enemies act as obstacles throughout the level
- Shooting enemies rewards the player with points
- Colliding with any enemy results in a game over
- Destroyable bonus boxes grant additional score
- A special Big Enemy requires multiple hits to defeat

## Scoring System
- Regular enemy defeated: +1 point
- Big Enemy defeated: +5 points
- Bonus boxes destroyed: +2 points

## Win & Lose Conditions
- Win: Reach the final scene of the level
- Lose: Collide with any enemy

## Features
- Complete single-player game loop (start ‚Üí play ‚Üí win/lose)
- Shooter mechanics
- Obstacle-based level design
- Multiple enemy types
- Simple scoring and progression system

## Configuration & Setup
- Engine: Unity
- Language: C#
- Platform: PC
- Input: Keyboard

To run the project:
1. Open the project using Unity Hub
2. Load the main scene
3. Press Play to start the game

## Project Context
This game was developed as part of a university game development course.  
The project focuses on implementing core gameplay mechanics and scoring logic.
